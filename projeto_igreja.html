Sistema de Gerenciamento de Membros da Igreja

Uma aplicação web completa para otimizar a administração e o engajamento de membros, oferecendo uma plataforma segura e intuitiva para gerenciar informações, facilitar a comunicação e capacitar administradores.

Principais Funcionalidades

Esta seção detalha os recursos essenciais que tornam o sistema uma ferramenta poderosa para a gestão da igreja. Cada funcionalidade foi projetada com foco na usabilidade e segurança.

Autenticação Segura

Sistema robusto de login e cadastro com recuperação de senha por e-mail para garantir o acesso seguro dos usuários.

Cadastro de Membros

Formulário intuitivo para novos membros, capturando detalhes essenciais, incluindo foto de perfil e endereço com integração de CEP.

Gerenciamento de Perfil

Páginas de perfil dedicadas onde os membros podem visualizar e atualizar suas informações pessoais a qualquer momento.

Painel Administrativo

Um painel de controle completo para administradores com listagem, busca e paginação de todos os membros.

Visualização de Fotos

Visualize as fotos dos membros diretamente na lista do painel, com uma opção de zoom para ver detalhes.

Integração com Firebase

Uso completo do ecossistema Firebase para autenticação, banco de dados em tempo real (Firestore) e armazenamento de arquivos (Storage).
